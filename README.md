# My_Health

Flutter ile geliştirilmiş, kullanıcıların sağlık bilgilerini takip edebildiği, ilaç hatırlatmalarını yönetebildiği ve kişisel sağlık verilerini görüntüleyebildiği modern bir mobil uygulama.

## Özellikler
- İlaç hatırlatma sistemi  
- Sağlık verisi ve tahlil takibi  
- Kullanıcı profili yönetimi  
- Modern ve sade arayüz

## Kullanılan Teknolojiler
- Flutter & Dart (Frontend)
- Spring Boot (Backend)  
- PostgreSQL (DB)  
- REST API
